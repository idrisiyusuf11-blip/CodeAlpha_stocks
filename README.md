# Task 2: Stock Portfolio Tracker

A tool to manage and calculate the value of your stock investments based on predefined market prices.

### 🚀 Features
* **Real-time Calculations:** Automatically calculates sub-totals and total portfolio value.
* **Data Export:** Option to save your final portfolio summary to a `.txt` file.
* **Tabular Display:** Clean, formatted table output in the console.

### 🛠️ Key Concepts
* Dictionaries (for stock prices)
* File I/O (Saving results)
* Error handling (ValueError for quantities)

### 🕹️ How to Run
1. Run: `python stocks.py`
2. Follow the prompts to add tickers like `AAPL` or `TSLA`.
