Project Description:
A console-based Stock Portfolio Tracker that allows users to input stock names and quantities, then calculates total investment based on predefined stock prices.


Features:
•Dictionary-based stock price storage
•User input for stock name and quantity
•Investment calculation
•Input validation
•Data saving to text file


Technologies Used:
•Python
•Dictionaries
•File Handling
•Arithmetic operations


How to Run:
python portfolio_tracker.py


Learning Outcomes:
•Dictionary manipulation
•File read/write operations
•Error handling and validation
